Chưa kết nối Sheet
Data đọc trực tiếp từ Google Sheet đã publish. Sửa Sheet → bấm "Tải lại" → dashboard cập nhật. Số liệu tính từ data thật, không có số giả.

Kết nối Google Sheet

Cách lấy link (làm 1 lần):
1. Trong Google Sheet: File → Share → Publish to web
2. Chọn từng sheet (OVERALL, LIVE, VIDEO), định dạng CSV
3. Bấm Publish, copy link (dạng .../pub?...output=csv)
4. Dán 3 link vào 3 ô dưới, bấm Kết nối & tải data
